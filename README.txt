Syntax highlighting by Matthew Tesch for the Intro-Logic course.

To install this file: 
	- Download the latest version and save it in a convenient location
	- Open Notepad++
	- In the top bar, click Language, then mouse over "User Defined Language", and click "Define your language..."
	- Click "Import", then open the file in the opened file prompt.
	
To use the language:
	- In the top bar, click Language, then select the entry near the bottom that reads "IntroLogic" followed by a version number.

To stop using the language:
	- In the top bar, click Language, then select "None (normal text)".
	
See the Wiki for some images on what this looks like!
